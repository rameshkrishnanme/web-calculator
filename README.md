# Premium Web Calculator

> **Note:** This project was 100% generated by Antigravity.

A modern, web-based calculator application built using vanilla HTML, CSS, and JavaScript, served via a Node.js/Express backend.  It features a sleek, premium "glassmorphism" aesthetic.

## Features & Implementation

*   **Backend Server:** A custom Node.js backend using Express to serve the static frontend efficiently on port `3000`.
*   **Premium Aesthetic:** Styled completely with vanilla CSS. Includes a dark mode backdrop with a translucent, blurred calculator body (glassmorphism), refined box-shadows, and smooth hover effects.
*   **Typography:** Utilizes Google's `Inter` font for a professional, highly readable display.
*   **Robust JavaScript Logic:** The calculator logic is encapsulated within a robust ES6 `Calculator` class. It flawlessly handles:
    *   Basic arithmetic operations (`+`, `-`, `×`, `÷`).
    *   Complex edge-cases (e.g., preventing division by zero).
    *   Responsive decimal alignment.
*   **Keyboard Support:** Full integration with the keyboard, allowing seamless calculation using keys like `Enter`, `Backspace`, `Escape` (for All Clear), and the standard numeric pad operators.

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Execution

1.  Navigate into the project directory:
    ```bash
    cd web-calculator
    ```
2.  Install dependencies (Express):
    ```bash
    npm install
    ```
3.  Start the local server:
    ```bash
    npm start
    ```
4.  Open your browser and navigate to:
    ```text
    http://localhost:3000
    ```
